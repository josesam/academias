Du kan välja från följande alternativ för att redigera SugarPortal:

  • Redigera en modul. Välj det här alternativet för att redigera Bug Tracker, Ärenden och Leads modulen.
  • Ladda upp ett stylesheet. Välj det här alternativet för att redigera default stylesheet för SugarPortal.
  • Portal Synk. Välj det här alternativet för att synkronisera SugarPortal med Sugar server.

För att redigera en modul, gör följande:

  1. Välj modulen som du vill redigera i SugarPortal.
  2. Välj en vy som du vill redigera.
    Portal redigeringsverktyget visar ett verktygsfält till vänster och fält vy detaljerna till höger. För att redigera Redigerings vyn eller list vyn klicka på länken ovan. Verktygsfältet visar tillgängliga fält som för tillfället inte finns i detalj vyn. Det visar också tomma samt blanka fält. Använd det blanka fältet för att skapa mellanslag eller tomma ytor i en vy. Till exempel, använd ett blankt fält bredvid Beskrivningen för att skapa tillräcklig mellanrum för att skilja till detaljerna åt. Använt ett tomt fält för att skapa en yta där du kan ersätta med ett riktigt fält i framtiden.
  3. För att lägga till ett fält från verktygsfältets tabell till detalj vyn, klicka Lägg till rad. Välj fält från verktygsfältet och dra ned det på det tomma fältet.
  4. För att ta bort ett fält från vyn, välj det och dra det till verktygsfältet.
  5. Klicka Spara för att spara ändringarna.

För att ladda upp ett stylesheet, gör följande:

  1. Välj Ärenden, Bug Tracker eller nyhetsbrev.
  2. Klicka Browse, navigera till platsen för ditt stylesheet och välj det.
  3. Klicka Ladda upp för att visa stylesheet.

För att synkronisera SugarPortal med Sugar server gör följande:

  1. Enter the URL to the portal instance that you want to update and click Go.
  2. Ange URL till portal instansen som du vill uppdatera och klicka Go.
  3. Ange användarnamn och lösenord samt klicka på Börja synk.
