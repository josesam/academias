Studio

Pode escolher uma das seguintes opções para editar o SugarPortal:

  • Editar um módulo. Seleccione esta opção para editar os módulos de Bug Tracker, Ocorrências ou Leads
  • Carregar uma Style Sheet. Seleccione esta opção para carregar uma stylesheet no SugarPortal.
  • Sincronizar o Portal. Seleccione esta opção para sincronizar o Portal com o Sugar.

Para editar um módulo, faça o seguinte:

  1. Seleccione o módulo que deseja reconfigurar no SugarPortal.
  2. Seleccione a vista que deseja alterar.
    O Editor do Portal vai exibir uma barra de ferramentas no lado esquerdo e os campos de detalhe à direita. Para alterar a vista de Edição ou Listagem, seleccione o link apropriado na parte superior do ecrã. A barra de ferramentas vai exibir os campos disponíveis que actualmente não estão na vista de detalhe. Também exibe um campo vazio e um campo em branco. Utilize o campo em branco para criar espaços na vista. Por exemplo, use o campo em branco ao lado da Descrição para reservar mais espaço para uma descrição detalhada. Use um campo vazio para reservar espaço no formulário para o futuro.
  3. Para acrescentar um campo da barra de ferramentas à vista de detalhe, clique em Adicionar Linha. Seleccione o campo da barra de ferramentas e arraste-o para o campo vazio.
  4. Para remover um campo da vista, seleccione-o, e arraste-o para a barra de ferramentas.
  5. Clique Gravar para gravar as suas alterações.

Para carregar uma stylesheet, faça o seguinte:

  1. Seleccione o tabulador para os módulos de Ocorrências, Bug Tracker, ou Newsletters.
  2. Click Browse, navegue até à localização da stylesheet, e seleccione-a.
  3. Click Upload para carregar a stylesheet.

Para sincronizar o Portal com o servidor de Sugar, faça o seguinte:

  1. Introduza o URL da instância do portal que quer actualizar e clique em Avançar.
  2. Insira o seu nome de utilizador e palavra-passe e clique em Iniciar Sincronização.
