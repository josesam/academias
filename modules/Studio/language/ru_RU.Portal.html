Студия

Доступные опции для редактирования Портала Sugar:

  • Изменить модуль. Выберите эту опцию для изменения модулей Ошибки, Обращения или Предварительные контакты.
  • Upload a Style Sheet. Выберите эту опцию для изменения стилевого оформления Портала Sugar по умолчанию.
  • Синхронизация Портала. Выберите эту опцию для синхронизации Портала Sugar с сервером Sugar.

Действия для изменения модуля:

  1. Выберите модуль, который необходимо перенастроить в портале Sugar.
  2. Выберите представление, которое хотите редактировать.
    Редактор Портала отображает панель инструментов слева и поля детального просмотра справа. Для перехода к форме редактирования или списку записей выберите соответствующую ссылку выше. Панель инструментов содержит доступные поля, которые в данный момент отсутствуют на форме детального просмотра. Она также содержит пустое поле и пустой ярлык для разделения записей. Используйте пустой ярлык для разделения записей для создания пустых областей при просмотре. Например, пустой ярлык для разделения записей рядом с Описанием может использоваться для создания пустого места для подробного описания. Пустой ярлык может использоваться как заполнитель свободного места, который в будущем можно заменить на имя поля.
  3. Для того, чтобы добавить поле из таблицы инструментов в поле детального просмотра, нажмите Добавить строку. Выберите поле из панели инструментов и перетяните его в пустое поле.
  4. Для удаления поля и детального просмотра, выберите его, и перетяните в панель инструментов.
  5. Нажмите Сохранить для сохранения изменений.

Действия для загрузки стилевого оформления:

  1. Выберите соответствующую вкладку: Обращения, Ошибки или Информационные бюллетени.
  2. Нажмите Открыть, выберите расположение файла стилевого оформления и выберите его.
  3. Нажмите Загрузить для отображения нового стилевого оформления.

Действия для синхронизации портала Sugar с сервером Sugar:

  1. Введите URL копии портала, которую необходимо обновить, и нажмите Вперед.
  2. Введите логин и пароль и нажмите Начать синхронизацию.
